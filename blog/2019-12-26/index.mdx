The trick is that when you use position absolute with top/right/bottom/left 0 it fills the container.

```
#wrapper {

margin:0;

position: relative;

height: 100%;

overflow: hidden;

}

#sidebar{

position: absolute;

top: 0;

left: 0;

bottom: 0;

height: 100%;

}
```
